QUALIPHY API DOCS · OBSERVABILITY DASHBOARD · SETUP
================================================================================

What this does
--------------
The pages at docs.qualiphy.me/ and docs.qualiphy.me/quick-start/ now ship a
small tracker block. Each visitor gets a persistent visitor ID stored in
localStorage; events the visitor generates (page load, section views, code
copies, search queries, tab switches, scroll depth, page unload, plus every
option they pick in the Quick Start plan builder) are posted fire-and-forget
to a Google Apps Script web app you control. The web app upserts a single
row per visitor into a Google Sheet.

The dashboard at docs.qualiphy.me/dashboard/ is password-gated. After login
it pulls the same Google Sheet through the same Apps Script (GET request
with a shared secret) and renders it Grafana-style.

Three things to wire up:
  (A) Create the Google Sheet
  (B) Paste the Apps Script + deploy it as a web app
  (C) Drop the deployed /exec URL into three places in the repo


================================================================================

================================================================================

If you already have a v1 sheet from the earlier deploy, you can reuse it
exactly as-is — the v2 script self-extends the header row, so missing

/**
 * Qualiphy API Docs telemetry endpoint (v3).
 *
 * doPost(e)  — receives events from the docs pages, upserts a row per visitor.
 *              Drops PII into the sheet for internal reference only.
 * doGet(e)   — returns all rows as JSON when ?action=data&secret=... matches.
 *              PII columns (Email, Company, Contact Name, Webhook URL Value,
 *              Exam IDs Value) are FILTERED OUT of the response so the
 *              dashboard never receives them over the wire. The sheet stays
 *              the source of truth for manual lookups; the dashboard only
 *              sees aggregate metrics.
 *
 * v3 adds (vs v2):
 *   - PII capture: Email, Company, Contact Name, Webhook URL Value, Exam IDs Value
 *   - Per-feature flags: Opt: Prefill, Opt: Attach Docs, Opt: Custom Redirect,
 *     Opt: Additional Data, Opt: Skip Pharmacy Confirm, Opt: Clinic Billed,
 *     Opt: Ship To Patient, Opt: Skip Address Confirm
 *   - doGet PII filter so the dashboard cannot accidentally render PII.
 *
 * v2 schema additions (carried over): Use Cases, Pharmacy Choice, Provider
 * Override, Invite Method, Optional Features, States Selected, States List,
 * Has Webhook URL, Has Exam IDs, Quick Start Step, Quick Start Completed,
 * Tab Languages.
 *
 * The shared secret here MUST match the secret used by:
 *   - q-api/index.html and q-api/quick-start/index.html (in the tracker block)
 *   - q-api/dashboard/index.html (in CONFIG.SHARED_SECRET)
 */

const OPT_FEATURE_COLS = {

function getSheet_() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  let sh = ss.getSheetByName(SHEET_NAME);
  if (!sh) {
    sh = ss.insertSheet(SHEET_NAME);
    sh.getRange(1, 1, 1, COLS.length).setValues([COLS]);
    sh.setFrozenRows(1);
  }
  ensureColumns_(sh);
  return sh;
}

/**
 * Extends the header row with any columns from COLS that aren't there yet.
 * Existing data and column positions are preserved; new columns append to
 * the right. This is what makes a v1→v2 redeploy non-destructive.
 */
function ensureColumns_(sh) {
  const lastCol = sh.getLastColumn();
  let header = lastCol > 0 ? sh.getRange(1, 1, 1, lastCol).getValues()[0].map(String) : [];
  const missing = COLS.filter(c => header.indexOf(c) === -1);
  if (!missing.length) return;
  if (lastCol === 0) {
    // Empty sheet - write the whole COLS row
    sh.getRange(1, 1, 1, COLS.length).setValues([COLS]);
    sh.setFrozenRows(1);
  } else {
    sh.getRange(1, lastCol + 1, 1, missing.length).setValues([missing]);
  }
}

function headerMap_(sh) {
  const lastCol = sh.getLastColumn();
  const header = sh.getRange(1, 1, 1, lastCol).getValues()[0].map(String);
  const map = {};
  header.forEach((name, i) => { if (name) map[name] = i + 1; });
  return { map: map, width: header.length };
}

function getRowMap_(sh, hMap) {
  const last = sh.getLastRow();
  const map = {};
  if (last < 2) return map;
  const sidCol = hMap.map['Session ID'] || 1;
  const ids = sh.getRange(2, sidCol, last - 1, 1).getValues();
  for (let i = 0; i < ids.length; i++) {
    const id = String(ids[i][0] || '').trim();
    if (id) map[id] = i + 2;
  }
  return map;
}

function readRow_(sh, rowNum, hMap) {
  const vals = sh.getRange(rowNum, 1, 1, hMap.width).getValues()[0];
  const obj = {};
  Object.keys(hMap.map).forEach(name => { obj[name] = vals[hMap.map[name] - 1]; });
  return obj;
}

function writeRow_(sh, rowNum, obj, hMap) {
  const row = new Array(hMap.width).fill('');
  Object.keys(obj).forEach(name => {
    if (hMap.map[name]) row[hMap.map[name] - 1] = obj[name];
  });
  sh.getRange(rowNum, 1, 1, hMap.width).setValues([row]);
}

function appendRow_(sh, obj, hMap) {
  const row = new Array(hMap.width).fill('');
  Object.keys(obj).forEach(name => {
    if (hMap.map[name]) row[hMap.map[name] - 1] = obj[name];
  });
  sh.appendRow(row);
}

function classifyUA_(ua) {
  ua = String(ua || '');
  let device = 'Desktop';
  if (/iPad|Tablet/i.test(ua)) device = 'Tablet';
  else if (/Mobile|iPhone|Android/i.test(ua)) device = 'Mobile';
  let browser = 'Other';
  if (/Edg\//i.test(ua)) browser = 'Edge';
  else if (/Chrome\//i.test(ua) && !/Edg\//i.test(ua)) browser = 'Chrome';
  else if (/Safari\//i.test(ua) && !/Chrome\//i.test(ua)) browser = 'Safari';
  else if (/Firefox\//i.test(ua)) browser = 'Firefox';
  return { device, browser };
}

function safeJson_(s, fallback) {
  if (!s) return fallback;
  if (typeof s === 'object') return s;
  try { return JSON.parse(s); } catch (e) { return fallback; }
}

function csvAdd_(curStr, value) {
  const list = String(curStr || '').split(',').map(s => s.trim()).filter(Boolean);
  if (list.indexOf(value) === -1) list.push(value);
  return list.join(',');
}
function csvRemove_(curStr, value) {
  return String(curStr || '').split(',').map(s => s.trim()).filter(Boolean).filter(v => v !== value).join(',');
}

function doPost(e) {
  let body = {};
  try {
    body = JSON.parse(e.postData.contents || '{}');
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ ok: false, error: 'bad json' }))
      .setMimeType(ContentService.MimeType.JSON);
  }
  if (body.secret !== SHARED_SECRET) {
    return ContentService.createTextOutput(JSON.stringify({ ok: false, error: 'unauthorized' }))
      .setMimeType(ContentService.MimeType.JSON);
  }
  const sid = String(body.session_id || '').trim();
  if (!sid) {
    return ContentService.createTextOutput(JSON.stringify({ ok: false, error: 'no session_id' }))
      .setMimeType(ContentService.MimeType.JSON);
  }

  const sh = getSheet_();
  const lock = LockService.getScriptLock();
  lock.waitLock(20000);
  try {
    const hMap = headerMap_(sh);
    const rowMap = getRowMap_(sh, hMap);
    const isNew = !(sid in rowMap);
    const rowNum = isNew ? sh.getLastRow() + 1 : rowMap[sid];
    const cur = isNew ? {} : readRow_(sh, rowNum, hMap);

    if (isNew) {
      cur['Session ID'] = sid;
      cur['Started At'] = body.timestamp || new Date().toISOString();
    }
    cur['Last Activity'] = body.timestamp || new Date().toISOString();

    if (body.path && !cur['Path']) cur['Path'] = body.path;
    if (body.referrer && !cur['Referrer']) cur['Referrer'] = body.referrer;
    if (body.user_agent && (!cur['Device'] || !cur['Browser'])) {
      const ua = classifyUA_(body.user_agent);
      cur['Device'] = cur['Device'] || ua.device;
      cur['Browser'] = cur['Browser'] || ua.browser;
    }
    if (body.utms) {
      if (body.utms.utm_source && !cur['UTM Source'])     cur['UTM Source']   = body.utms.utm_source;
      if (body.utms.utm_medium && !cur['UTM Medium'])     cur['UTM Medium']   = body.utms.utm_medium;
      if (body.utms.utm_campaign && !cur['UTM Campaign']) cur['UTM Campaign'] = body.utms.utm_campaign;
    }
    if (body.geo) {
      if (body.geo.city && !cur['City'])       cur['City']    = body.geo.city;
      if (body.geo.region && !cur['State'])    cur['State']   = body.geo.region;
      if (body.geo.country && !cur['Country']) cur['Country'] = body.geo.country;
    }
    if (body.variant && !cur['Variant']) cur['Variant'] = body.variant;

    cur['Total Events'] = (Number(cur['Total Events']) || 0) + 1;

    const ev = String(body.event || '');
    const data = body.data || {};

    if (ev === 'section_viewed' && data.section_id) {
      const list = String(cur['Sections List'] || '').split(',').map(s => s.trim()).filter(Boolean);
      if (list.indexOf(String(data.section_id)) === -1) {
        list.push(String(data.section_id));
        cur['Sections List'] = list.join(',');
        cur['Sections Viewed'] = list.length;
      }
      cur['Last Section'] = String(data.section_id);
    }
    else if (ev === 'code_copied' && data.pane_id) {
      cur['Code Copies'] = (Number(cur['Code Copies']) || 0) + 1;
      const copies = safeJson_(cur['Top Copies'], {});
      copies[String(data.pane_id)] = (Number(copies[String(data.pane_id)]) || 0) + 1;
      cur['Top Copies'] = JSON.stringify(copies);
    }
    else if (ev === 'search_query' && data.query) {
      cur['Search Queries'] = (Number(cur['Search Queries']) || 0) + 1;
      const terms = safeJson_(cur['Search Terms'], []);
      const term = String(data.query).slice(0, 80);
      if (terms[terms.length - 1] !== term) terms.push(term);
      while (terms.length > 50) terms.shift();
      cur['Search Terms'] = JSON.stringify(terms);
    }
    else if (ev === 'tab_switched') {
      cur['Tab Switches'] = (Number(cur['Tab Switches']) || 0) + 1;
      if (data.language) {
        const langs = safeJson_(cur['Tab Languages'], {});
        langs[String(data.language)] = (Number(langs[String(data.language)]) || 0) + 1;
        cur['Tab Languages'] = JSON.stringify(langs);
      }
    }
    else if (ev === 'external_link_clicked') {
      cur['External Clicks'] = (Number(cur['External Clicks']) || 0) + 1;
    }
    else if (ev === 'scroll_depth' && typeof data.pct === 'number') {
      cur['Scroll Max'] = Math.max(Number(cur['Scroll Max']) || 0, Math.round(data.pct));
    }
    else if (ev === 'page_unloaded') {
      if (typeof data.total_duration_sec === 'number') {
        cur['Duration (sec)'] = Math.max(Number(cur['Duration (sec)']) || 0, Math.round(data.total_duration_sec));
      }
    }
    else if (ev === 'location_resolved') {
      if (data.city && !cur['City'])       cur['City']    = data.city;
      if (data.region && !cur['State'])    cur['State']   = data.region;
      if (data.country && !cur['Country']) cur['Country'] = data.country;
    }
    /* ----- Quick Start plan-builder events ----- */
    else if (ev === 'option_selected') {
      const g = String(data.group || ''), v = String(data.value || '');
      if (g === 'useCase')               cur['Use Cases']         = csvAdd_(cur['Use Cases'], v);
      // Pharmacy is multi-select on the page (user can pick both partner AND
      // custom). Treat as csv so the second pick doesn't overwrite the first.
      else if (g === 'pharmacy')         cur['Pharmacy Choice']   = csvAdd_(cur['Pharmacy Choice'], v);
      else if (g === 'providerOverride') cur['Provider Override'] = v;
      else if (g === 'invite')           cur['Invite Method']     = v;
      else if (g === 'optionalFeature') {
        cur['Optional Features'] = csvAdd_(cur['Optional Features'], v);
        if (OPT_FEATURE_COLS[v]) cur[OPT_FEATURE_COLS[v]] = 'Yes';
      }
    }
    else if (ev === 'option_deselected') {
      const g = String(data.group || ''), v = String(data.value || '');
      if (g === 'useCase')               cur['Use Cases']         = csvRemove_(cur['Use Cases'], v);
      else if (g === 'pharmacy')         cur['Pharmacy Choice']   = csvRemove_(cur['Pharmacy Choice'], v);
      else if (g === 'optionalFeature') {
        cur['Optional Features'] = csvRemove_(cur['Optional Features'], v);
        if (OPT_FEATURE_COLS[v]) cur[OPT_FEATURE_COLS[v]] = '';
      }
      // providerOverride / invite are single-choice, not deselected
    }
    else if (ev === 'qs_states_changed') {
      cur['States Selected'] = Number(data.count) || 0;
      if (data.list !== undefined) cur['States List'] = String(data.list).slice(0, 500);
    }
    else if (ev === 'qs_step_advanced') {
      cur['Quick Start Step'] = Math.max(Number(cur['Quick Start Step']) || 0, Number(data.to) || 0);
      if (Number(data.to) >= 4) cur['Quick Start Completed'] = 'Yes';
    }
    else if (ev === 'qs_step_back') {
      // Don't lower the high-water mark, but record we went back if useful.
    }
    else if (ev === 'qs_completed') {
      cur['Quick Start Step'] = 4;
      cur['Quick Start Completed'] = 'Yes';
    }
    else if (ev === 'qs_field_filled') {
      if (data.field === 'webhook') cur['Has Webhook URL'] = 'Yes';
      if (data.field === 'examIds') cur['Has Exam IDs']    = 'Yes';
    }
    /* v3 - actual PII values, sheet-only, never returned to dashboard */
    else if (ev === 'qs_field_value') {
      const f = String(data.field || ''), v = String(data.value || '').slice(0, 500);
      if (f === 'email')        cur['Email']             = v;
      else if (f === 'company') cur['Company']           = v;
      else if (f === 'name')    cur['Contact Name']      = v;
      else if (f === 'webhook') cur['Webhook URL Value'] = v;
      else if (f === 'examIds') cur['Exam IDs Value']    = v;
    }
    else if (ev === 'qs_restart') {
      // Wipe plan-builder fields so the same visitor can re-answer fresh.
      cur['Use Cases'] = '';
      cur['Pharmacy Choice'] = '';
      cur['Provider Override'] = '';
      cur['Invite Method'] = '';
      cur['Optional Features'] = '';
      cur['States Selected'] = '';
      cur['States List'] = '';
      cur['Has Webhook URL'] = '';
      cur['Has Exam IDs'] = '';
      cur['Quick Start Step'] = '';
      cur['Quick Start Completed'] = '';
      // v3 - also wipe PII + opt flags so the row reflects the fresh attempt.
      // The PDF Drive URL is NOT wiped: the file still exists in Drive and the
      // URL is useful as a historical pointer even if the user re-runs the flow.
      cur['Email'] = '';
      cur['Company'] = '';
      cur['Contact Name'] = '';
      cur['Webhook URL Value'] = '';
      cur['Exam IDs Value'] = '';
      Object.keys(OPT_FEATURE_COLS).forEach(function(k){ cur[OPT_FEATURE_COLS[k]] = ''; });
    }

    if (typeof body.elapsed_sec === 'number') {
      cur['Duration (sec)'] = Math.max(Number(cur['Duration (sec)']) || 0, Math.round(body.elapsed_sec));
    }

    if (isNew) appendRow_(sh, cur, hMap);
    else writeRow_(sh, rowNum, cur, hMap);

    return ContentService.createTextOutput(JSON.stringify({ ok: true }))
      .setMimeType(ContentService.MimeType.JSON);
  } finally {
    lock.releaseLock();
  }
}

function doGet(e) {
  const params = (e && e.parameter) || {};
  if (params.secret !== SHARED_SECRET) {
    return ContentService.createTextOutput(JSON.stringify({ error: 'unauthorized' }))
      .setMimeType(ContentService.MimeType.JSON);
  }
  if (params.action !== 'data') {
    return ContentService.createTextOutput(JSON.stringify({ ok: true, hint: 'use ?action=data&secret=...' }))
      .setMimeType(ContentService.MimeType.JSON);
  }
  const sh = getSheet_();
  const hMap = headerMap_(sh);
  const last = sh.getLastRow();
  const sessions = [];
  // v3: filter out PII columns at the API boundary. Dashboard never sees
  // these values, so it can't accidentally render them. Sheet keeps them.
  const piiSet = {}; PII_COLS.forEach(function(c){ piiSet[c] = 1; });
  if (last >= 2) {
    const vals = sh.getRange(2, 1, last - 1, hMap.width).getValues();
    const headerNames = Object.keys(hMap.map).sort((a, b) => hMap.map[a] - hMap.map[b]);
    vals.forEach(row => {
      const obj = {};
      headerNames.forEach(name => {
        if (piiSet[name]) return; // skip PII
        let v = row[hMap.map[name] - 1];
        if (v instanceof Date) v = v.toISOString();
        obj[name] = v;
      });
      sessions.push(obj);
    });
  }
  return ContentService.createTextOutput(JSON.stringify({
    ok: true,
    updated: new Date().toISOString(),
    sessions: sessions
  })).setMimeType(ContentService.MimeType.JSON);
}

LOOKUP WORKFLOW
---------------
When you need a clinic's full intake details (e.g. preparing for a
follow-up call):
  1. Open the Sheet, hit Ctrl/Cmd+F, paste their email.
  2. Jump to the matching row.
  3. Scroll right — Use Cases, Pharmacy Choice, Provider Override, Invite
     Method, Optional Features (csv + per-feature flags), States, Webhook
     URL, Exam IDs are all on that row.
The PDF the user downloaded is generated client-side from these exact
fields, so reconstructing what they would have seen is straightforward
without storing the binary.

PII FILTER ON doGet
-------------------
doGet() iterates row values and skips any column listed in PII_COLS. The
dashboard never sees Email / Company / Contact Name / Webhook URL Value /
Exam IDs Value over the wire — meaning they can't accidentally render in
the recent-sessions table, can't be included in the CSV export, and aren't
sniffable from the browser dev tools.

The sheet itself, of course, has all of this for your manual reference.

F) SMOKE TESTING THAT EVERYTHING WORKS
================================================================================

1. After (re)deploying the script, open any docs page. Within ~1–2 seconds
   a row should appear (or update) in your Sheet. The "Started At", "Path",
   "Referrer", "Device", "Browser", and "UTM Source" columns populate
   immediately.

2. In the docs page, click around — view a few endpoints, click Copy on
   a code block, type something into the search box. Watch your row in
   the sheet update:
     - "Sections Viewed" / "Sections List" grows as you scroll past sections
     - "Code Copies" / "Top Copies" increments on each Copy click
     - "Search Queries" / "Search Terms" increments per debounced query
     - "Tab Languages" tracks which curl/node/python tabs you toggle

3. Visit /quick-start/ and answer the questionnaire. As you click options:
     - "Use Cases" fills in (e.g. "gfe,rx")
     - "Pharmacy Choice", "Provider Override", "Invite Method" populate
     - "Optional Features" tracks the checkboxes
     - "States Selected" / "States List" track the state grid
     - "Quick Start Step" climbs as you advance
     - "Quick Start Completed" flips to "Yes" when you generate the plan

4. Visit /dashboard/. Enter qualiphyapi2026. The new "Quick Start Plan
   Intent" panels should populate from your test session.

5. If the dashboard shows "no webhook" or "err" next to the live indicator,
   you forgot step C.1 (the dashboard's CONFIG.WEBHOOK_URL still has the
   placeholder).


================================================================================
G) TROUBLESHOOTING
================================================================================

"unauthorized" in the doGet response
  The dashboard's CONFIG.SHARED_SECRET doesn't match SHARED_SECRET in the
  Apps Script. Both should be 'qapi-9j2k-trk-bx88-2026'. Check both ends.

Tracker doesn't write rows when I visit the page
  Open browser DevTools > Network, reload the page, look for a request to
  script.google.com/macros/.../exec. If it 404s, your /exec URL is wrong.
  If it 200s but no row appears, double-check that the SHARED_SECRET in
  the tracker block matches the script.

Geo columns stay blank
  The tracker tries three free IP-geolocation services (geojs, ipapi,
  freeipapi). If none responds in time, geo stays blank. This is normal
  and shouldn't block any other tracking.

I want to start fresh / clear the data
  Open the Sheet, select all rows below row 1, right-click > Delete rows.
  Or rename the "Sessions" tab — the script will create a new empty tab
  on the next event.

I want to rotate the secret
  1. Edit SHARED_SECRET in Code.gs and redeploy (Deploy > Manage deployments
     > pencil icon > new version).
  2. Edit CONFIG.SHARED_SECRET in q-api/dashboard/index.html.
  3. Edit QAPI_TRACK_CONFIG.SHARED_SECRET in q-api/index.html and
     q-api/quick-start/index.html.

I want to remove the tracker entirely
  In each of the three files, delete the script block bounded by the
  comments:
    /* ============ Q-API TRACKER · BEGIN ============ */
    ...
    /* ============ Q-API TRACKER · END   ============ */
  And delete q-api/dashboard/ entirely.
